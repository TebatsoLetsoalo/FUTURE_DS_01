This folder contains the Power BI dashboard developed for the Business Sales Performance Analysis project.

The dashboard provides an interactive view of key business metrics, including:

-Total Sales (Revenue)

-Total Profit

-Profit Margin (%)

-Sales trends over time

-Product performance

-Regional performance

The dashboard enables users to explore data through filters and visualizations, helping to identify trends, high-performing products, and areas requiring improvement.
